## Step 3: Theme Upload Karein

### Method 1: Via Admin Panel

1. **Admin Panel Mein Jayein**
   ```
   Online Store > Themes
   ```

2. **Theme Upload Karein**
   - "Add theme" button click karein
   - "Upload zip file" select karein
   - `shopify-theme.zip` file select karein
   - Upload karein

3. **Theme Preview**
   - Upload ke baad theme list mein dikhega
   - "Actions" > "Preview" click karein
   - Full theme test kar sakte hain

### Method 2: Via Shopify CLI (Advanced)

```bash
# Install Shopify CLI
npm install -g @shopify/cli @shopify/theme

# Theme folder mein jayein
cd "/home/moiz/Documents/Shopify Theme"

# Login aur push
shopify theme push
```

## Step 4: Store Setup (Basic)

### Essential Settings:

1. **Store Information**
   - Settings > Store details
   - Store name, email, phone update karein

2. **Payment Methods** (Regular Store Only)
   - Settings > Payments
   - Payment provider setup karein

3. **Shipping** (Regular Store Only)
   - Settings > Shipping
   - Shipping zones configure karein

4. **Products Add Karein** (Testing Ke Liye)
   - Products > Add product
   - At least 2-3 test products add karein
   - Images add karein
   - Prices set karein

5. **Collections Banayein**
   - Products > Collections
   - "Create collection" click karein
   - Products assign karein

6. **Navigation Menu**
   - Online Store > Navigation
   - Main menu create karein
   - Collections add karein

---

## Step 5: Theme Customization

1. **Theme Settings**
   - Online Store > Themes
   - Uploaded theme ke saamne "Customize" click karein

2. **Colors & Fonts**
   - Theme settings > Colors
   - Primary, Secondary, Accent colors set karein
   - Typography > Fonts choose karein

3. **Header Setup**
   - Theme settings > Header
   - Logo upload karein ya text enter karein
   - Menu select karein

4. **Homepage Sections**
   - Customize page mein sections add/edit karein
   - Hero banner configure karein
   - Featured collection select karein
