## Installation

1. Zip the entire theme folder
2. Go to your Shopify admin
3. Navigate to Online Store > Themes
4. Click "Add theme" > "Upload zip file"
5. Select your theme zip file
6. Click "Upload"

## Customization

### Colors
- Go to Theme Settings > Colors
- Customize Primary, Secondary, and Accent colors

### Typography
- Go to Theme Settings > Typography
- Choose your heading and body fonts

### Header
- Go to Theme Settings > Header
- Add your logo image or text
- Configure your menu

### Footer
- Go to Theme Settings > Footer
- Customize footer text

## Sections

### Homepage Sections
- **Hero Banner**: Large banner with heading, text, and CTA button
- **Featured Collection**: Display products from a selected collection
- **Featured Products**: Display specific featured products

## Templates

- **Index**: Homepage template
- **Product**: Individual product page
- **Collection**: Collection/listing page
